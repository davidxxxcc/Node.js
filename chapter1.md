Open CMD

#### Check your node.js version

```
node -v
```

#### Use node.js directly from the CMD

```
node
```

#### Exit the node.js mode

```
process.exit(0) or Ctrl + c
```

#### Check the window object in node.js mode

```
global
```

#### Check the node process in node.js mode

```
process
```
